MapCreateLockedCopyPacket (id: 131)

Purpose: This is fired when the user locks a map item utilizing the Cartography Table in game.
Notes: It sends the original map id and the new map id. On the server it follows a similar process to creating a new map, sends the data and the map info to the client.

Payload




Field NameField TypeField ConstraintsField Notes
Original Map Id ActorUniqueID Id of the map being locked.
New Map Id ActorUniqueID Id that the new map should have.
