CurrentStructureFeaturePacket (id: 314)

Purpose: Informs the client of which Structure Feature they are currently occupying.
Notes: Sends the name of the Structure Feature the player is currently occupying to the client. If the player is not in a structure, this packet contains an empty string.

Payload




Field NameField TypeField ConstraintsField Notes
Current Structure Feature string The identifier of the Structure Feature that the player is currently occupying. If the player is not occupying a structure then this value is an empty string.
