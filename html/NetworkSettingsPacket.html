NetworkSettingsPacket (id: 143)

Purpose: Sends tunable options from host to client (compression threshold and algorithm)
Notes:

Payload




Field NameField TypeField ConstraintsField Notes
Compression Threshold unsigned short Determines the smallest size of raw network payload to compress.
NOTE: 0 is disable compression, 1 is compress everything 1 byte or larger (so everything)
CompressionAlgorithm enum PacketCompressionAlgorithm
Client Throttle Enabled bool
Client Throttle Threshold unsigned byte
Client Throttle Scalar float
